## 关机

```shell
shutdown : 关机
sync : 同步，加快内存到磁盘的速度
 who : 查看哪个用户在线
```

## 指令和文件搜索

``` shell
which filename 显示可执行文件的二进制文件
	which ping
		/bin/ping
	which netcat uptime
		/bin/netcat
		/usr/bin/uptime
whereis filename 定位命令的二进制文件，源文件和手册
	whereis bash
		bash: /bin/bash /etc/bash.bashrc /usr/share/man/man1/bash.1.gz
```
